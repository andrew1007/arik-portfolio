BattleBoats

With two Uno32 Pic kits I designed a game replicating the licensed Battleships board game. Written in C, I had the user play against a bot and using encrypted bitstrings and checksums to send back and forth the two kits during the game's progression. The bot utilizes random functions for all its guesses and the player sends bitstrings for their guesses. Each kit holds data for ship lives and locations

Arduino Mod GC Controller

I soldered a Arduino NANO 3.0 into my personal Gamecube controller. The Arduino contains custom code that fixes inconsistencies in inputs and adds a few features for the controller. The code specifically addresses many controller issues for the game Super Smash Bros Melee. The mod bridges the gap between bad and good controllers and balances the competetive playing field.
